## 🚀 GitHub Pages 배포 방법

1. **GitHub 저장소로 이동**
   - `https://github.com/partnerz2024/partnerzstock` 접속

2. **파일 업로드**
   - "Upload files" 버튼 클릭
   - **모든 파일**을 드래그 앤 드롭
   - 커밋 메시지: `Fix: Update all file paths to relative paths for GitHub Pages compatibility`
   - "Commit changes" 클릭

3. **GitHub Pages 설정 확인**
   - Settings → Pages
   - Source: "Deploy from a branch" 선택
   - Branch: "main" 선택
   - Folder: "/ (root)" 선택
   - Save 클릭
